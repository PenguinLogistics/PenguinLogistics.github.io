六种数据类型

基础数据类型

  • String

    • “String” || ‘String’
    • \转义字符

  • Number

    • typeof()方法查询变量类型
    • JS中最大数值 Number.MAX_VALUE
      如果超过最大数值，则等于数字(typeof返回值为Number)Infinity
    • NaN是一个特殊的数字，表示Not A Number
    • JS最小正值 Number.MIN_VALUE
    • 如果使用JS进行浮点数运算，可能得到一个不精确的值，原因是二进制无法精确表示1/10

  • Boolean

    • true/false

  • Null

    • Null类型的值只有一个，就是null，表示一个为空的对象
    • 用typeof检查,返回值为object

  • Undefined

    • Undefined类型值只有一个,就是undefined,表示未定义的值
    • 当声明一个变量,并不给变量赋值时,它的值是undefined
    • typeof的返回值为undefined
强制转型(转为String Number Boolean)方法

  • 转String

    1. 调用被转化数据类型的toString()方法

      • 该方法不会影响到原变量,他会将转换的结果返回
      • null和undefined两个值没有toString方法,如果调用,会报错

    2. 调用String()函数

      • 对于Number和Boolean实际上就是调用的toString()方法
        将null直接转换为”null”
        将undefined直接转换为”undefined”

  • 转Number

    1. 使用Number()//同String()
